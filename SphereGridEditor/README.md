# Sphere Grid Editor

Simple MonoGame-based visualizer for the sphere grid definition.

## Usage

```powershell
dotnet run --project .\SphereGridEditor\
```

## Controls

- **Left-click** - Select node (shows cost, powerup, connections)
- **Middle-click drag** - Pan camera
- **ESC** - Exit

## Visual Guide

- **Gold node** - Root
- **Light blue** - Has powerup
- **Gray** - Empty node
- **White/highlighted** - Selected/hovered
